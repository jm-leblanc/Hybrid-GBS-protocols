ABSOLUTE DUMMIES GUIDE TO USING COMPUTE CANADA AND DOING BIOINFORMATICS


#### BASIC LOGIN AND NAVIGATION

to access, open up a command line interface

```ssh username@cedar.computecanada.ca```

enter password when prompted
select appropriate 2FA when prompted


you're now in the "lobby" of compute Canada

### BASIC COMMANDS

ls : lists all files in the current directory
you can specify though, to look in different folders

ls folder1 will show what's in folder1 of the current directory

you can chain ../ in order to interact with folders and files in higher levels

cd ../ will take you to the parent directory (i.e. one level up)
cd ../../ will take you two levels up, and so on

you can ls ../otherfolder for example to see stuff without having to go to that folder

note: the commands are always in the context of your current working directory
so, if you're in /home/def-mcfarlas/projects/fescue

and fescue contains 'folder1', and 'folder2'

you can do something like ls folder1/data/barcodes to see that the file 'barcodes.txt' is found in that folder

then go cp folder1/data/barcodes/barcodes.txt .
that period indicates that you want to move that file to the current directory (i.e. fescue)

you could also go cp folder1/data/barcodes/barcodes.txt ../
to move the file to fescue's parent directory (i.e. projects)

cd: sets the directory to something new

mv: move something mv filename.ext targetdirectory/
or, if it's in a different folder, mv filename.ext ../otherfolder/

the trailing slash (e.g. folder1/) can be a good habit, it makes your command clear that you are specifying a folder or directory, in case there's a file with the same name.

you can also use mv to rename things

mv oldname newname



ctrl + C will interrupt the previous command if it's taking too long
