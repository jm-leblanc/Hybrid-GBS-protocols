## Pull out sample names corresponding to each of my two species

Allocate an interactive session, then load up R in compute canada

```bash
admixture <- read.csv("fescue_reassignment_CIs.csv")

# Check the structure to make sure it loaded correctly
head(admixture)
table(admixture$ci_geno_species)

# Extract sample names for FH species
fh_samples <- admixture$sample_name[admixture$ci_geno_species == "FH"]

# Extract sample names for FC species  
fc_samples <- admixture$sample_name[admixture$ci_geno_species == "FC"]

# Check how many samples you have
cat("FH samples:", length(fh_samples), "\n")
cat("FC samples:", length(fc_samples), "\n")

# Write to files for vcftools
write.table(fh_samples, "fh_samples.txt", row.names=FALSE, col.names=FALSE, quote=FALSE)
write.table(fc_samples, "fc_samples.txt", row.names=FALSE, col.names=FALSE, quote=FALSE)
```

Now I have lists that contain only the samples belonging to my parental populations (as determined by my confidence interval admixture overlaps)

# Calcluate FST between FH and FC

vcftools --gzvcf fescue_biallelic_snps_only.recode.vcf \
--weir-fst-pop fh_samples.txt \
--weir-fst-pop fc_samples.txt \
--out fh_vs_fc_genome_wide

This will give you an instant readout of the weighted FST (there might be a bunch of warnings before this text)
